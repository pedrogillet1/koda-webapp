# Chat Guide

The chat is where you interact with your documents using natural language.

## Key Features

- **Natural Language**: Ask questions in plain language
- **Cited Answers**: Get answers with source references
- **Multi-document**: Query across all your documents
- **Conversation History**: Access previous chats anytime

## How to Ask Questions

### Basic Questions
- "What is the main conclusion of the report?"
- "Summarize the key points"
- "What are the financial highlights?"

### Specific Queries
- "What does section 3.2 say about costs?"
- "Find information about the Q4 revenue"
- "Compare the two proposals"

### Document-Specific
- "In the contract, what are the payment terms?"
- "What does the Excel file show for 2024?"

## Understanding Responses

- **Bold text**: Key information
- **Citations**: Click to see the source document
- **Bullet points**: Organized key facts

## Tips for Better Results

1. **Be Specific**: Instead of "tell me about sales", ask "what were the Q3 2024 sales figures?"
2. **Reference Documents**: Mention specific file names when needed
3. **Follow Up**: Ask clarifying questions to dig deeper
4. **Use Context**: The AI remembers your conversation

## Starting New Conversations

- Click "New Chat" to start fresh
- Previous conversations are saved in history
- Each conversation maintains its own context
