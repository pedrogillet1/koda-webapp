# Documents Screen Guide

The documents screen lets you manage all your uploaded files and folders.

## Key Features

- **Folder Organization**: Create folders to organize your documents
- **Document Preview**: Quick preview without downloading
- **Search & Filter**: Find documents by name, type, or date
- **Bulk Actions**: Select multiple files for batch operations

## How to Use

### Creating Folders
1. Click the **{{btn.new_folder}}** button
2. Enter a name for your folder
3. Click create to confirm

### Managing Documents
- **View**: Click on any document to open it
- **Move**: Drag and drop to organize into folders
- **Delete**: Select and click **{{btn.delete}}**
- **Rename**: Right-click and select **{{btn.rename}}**

## File Types Supported

- PDF documents
- Word documents (.docx)
- Excel spreadsheets (.xlsx)
- PowerPoint presentations (.pptx)
- Images (PNG, JPG, JPEG)
- Text files (.txt)

## Tips

- Use descriptive folder names for easy navigation
- Archive old documents to keep your workspace clean
- Use the search bar to quickly find specific files
